## 🧐 About The Project <a name="about"></a>

![Demo Image](src/web-ui/public/images/demo.png/?raw=true 'Demo')

Web application for listing, creating, updating, and deleting bands, cafes, and gigs.

## ⭐️ Features <a name="features"></a>

- Login & Register
- CRUD for Bands
- CRUD for Cafes
- CRUD for Gigs
